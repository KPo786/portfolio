Feature: Make A Payment

User Story:
As a user,
I want to be able to make payments,
So that I can instantly transfer money to beneficiaries once I have successfully logged in, have set up my alias, my account is in good standing and I have sufficient funds available.
Scenario: User makes a payment for the first time When the user selects the option to add an alias, they can capture their alias, cellphone number, max payment amount, and default reference. The system validates field inputs and provides options to save or clear data with appropriate confirmation messaging.
Scenario: User makes a payment to a beneficiary Given the user's account is in good standing with sufficient funds, they can select a recipient with prepopulated cellphone number, enter payment amount, edit the default reference, and confirm payment. The system processes the instant payment transaction seamlessly.
